# Icons

Inline-able stroke SVGs using `stroke="currentColor"` so they take the
color of their context (e.g. active tab turns `--c-primary`).

These files are the **source library**. In screens, paste the `<svg>…</svg>`
inline (not via `<img>`) so `currentColor` and CSS sizing apply.

| File | Used by |
| --- | --- |
| `tab-today.svg`     | Today tab (home) |
| `tab-exercises.svg` | Exercises tab |
| `tab-progress.svg`  | Progress tab |
| `tab-rewards.svg`   | Rewards tab |
| `check.svg`         | done states, checklist ticks |
| `phone.svg`         | "Call the office" |
| `mic.svg`           | doctor-note voice input |
| `gear.svg`          | Today → Settings |
| `back.svg`          | back navigation |
| `eye.svg`           | "doctor can see" accountability line |
